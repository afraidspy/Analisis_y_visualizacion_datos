# Análisis y Visualización de Datos con Python

Este repositorio contiene material introductorio y práctico para el análisis y visualización de datos utilizando Python. A lo largo de los diferentes módulos se presentan las principales herramientas y técnicas utilizadas en ciencia de datos para manipular, transformar, analizar y visualizar información.

## Temario

### 1. NumPy y sus aplicaciones
- Introducción a NumPy
- Arreglos unidimensionales y multidimensionales
- Operaciones matemáticas y estadísticas
- Indexación y slicing
- Broadcasting
- Casos de uso en análisis de datos

### 2. Pandas y sus aplicaciones
- Introducción a Pandas
- Lectura y escritura de archivos
- Series y DataFrames
- Exploración inicial de datos
- Limpieza y transformación de información
- Aplicaciones en análisis de datos

### 3. Series y DataFrames
- Creación de Series
- Creación de DataFrames
- Atributos y métodos principales
- Selección de filas y columnas
- Tipos de datos
- Valores faltantes

### 4. DataFrames I: Operaciones básicas
- Visualización de datos
- Selección de columnas
- Renombrar columnas
- Crear nuevas variables
- Ordenamiento de datos
- Estadísticas descriptivas

### 5. DataFrames II: Filtrar datos
- Filtrado por condiciones
- Operadores lógicos
- Uso de `loc` e `iloc`
- Consultas complejas
- Filtrado de valores nulos
- Filtrado por texto

### 6. DataFrames III: Extraer datos
- Selección de subconjuntos
- Extracción por índices
- Métodos `head()`, `tail()` y `sample()`
- Uso de máscaras booleanas
- Extracción de información específica

### 7. DataFrames IV: Multi-índices y agrupación (GroupBy)
- Introducción a MultiIndex
- Creación y manipulación de índices jerárquicos
- Operaciones con `groupby()`
- Funciones de agregación
- Transformación y filtrado de grupos
- Resúmenes estadísticos

### 8. DataFrames V: Unir y concatenar (Merge/Join/Concat)
- Concatenación de DataFrames
- Uniones con `merge()`
- Uso de `join()`
- Tipos de joins:
  - Inner Join
  - Left Join

- Integración de múltiples fuentes de datos

### 9. Visualización con Matplotlib y Seaborn
- Introducción a Matplotlib
- Gráficas de líneas

- Histogramas
- Diagramas de dispersión
- Personalización de gráficos

- Comparación entre Matplotlib y Seaborn
